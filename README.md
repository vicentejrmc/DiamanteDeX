# Diamante De X

## Descrição

- **O Programa foi desenvolvido para funcionar no modo ConsoleApp** Para o Projeto a linguaguem ultilizada foi o C#.

O programa recebe um número inteiro positivo ímpar `n` e imprime um diamante de tamanho `n` com a letra `X`.

## Exemplo

Para `n = 5`, a saída deve ser:
```
   X
  XXX
 XXXXX
  XXX
   X
  ```
